- JavaScript
- Inertia.js
- Laravel
- MySQL
- React.js
- React Icons
- React Hot Toast

- Tailwind.css
- Video.js

## Installation

Clone and move to the repository

# important
don't forget to setup your mysql and .env

```bash
git clone https://github.com/AaronFabian/orange-juice.git;
cd orange-juice;
```

# install packages
```bash
composer install;
npm i;
```

# create the database
```bash
php artisan migrate:fresh;
```

# run the server
backend
```bash
php artisan serve;
```

frontend
```bash
npm run dev;
```
