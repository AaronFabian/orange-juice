## Installation

Clone and move to the repository

```bash
git clone https://github.com/AlvonJ/the-wild-oasis
cd the-wild-oasis

# install packages
composer install
npm i

# run the server
php artisan serve
npm run dev

# important
don't forget to setup you mysql and .env
```
